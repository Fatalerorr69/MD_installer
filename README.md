### Version Manager – Backup & Switch System
Umožňuje ukládat starší verze, obnovovat je a přepínat mezi nimi.

#### Záloha aktuální verze:
./version_manager/backup.sh

#### Výpis dostupných verzí:
./version_manager/switch.sh list

#### Aktivace konkrétní verze:
./version_manager/switch.sh use <verze>
